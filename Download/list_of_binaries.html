List of binaries available for NCL version 6.6.2

This is a full list of all the available precompiled binaries for NCL version 6.6.2 that are hosted on the Climate Data Gateway. If you don't see an appropriate binary for your system, then you may want to try a conda installation.

Go to the main download page for information on downloading and installing NCL.

Note that there are both OPeNDAP and non-OPenDAP enabled binaries. If you don't need OPeNDAP capabilities, then we recommend downloading an non-OPeNDAP binary, if available.

We are no longer including Cygwin binaries for Windows systems. We recommend that you use the Linux Bash shell under Windows to install NCL.

If you have a Linux system and are not sure which Linux binary to download, we recommend trying one of these two binary files if you need OPeNDAP support:

ncl_ncarg-6.6.2-CentOS6.10_64bit_gnu447.tar.gz
ncl_ncarg-6.6.2-Debian7.11_64bit_gnu472.tar.gz

or

ncl_ncarg-6.6.2-CentOS6.10_64bit_nodap_gnu447.tar.gz
ncl_ncarg-6.6.2-Debian7.11_64bit_nodap_gnu472.tar.gz

if you don't. For further information about Linux, see these Linux notes.

For convenience, we've provided a direct link to each gzipped tar file on the Climate Data Gateway so you can download it directly from this page.
