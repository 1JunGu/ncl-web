What's new in the application examples

24 February 2019

Added a new page that documents three routines added in NCL V6.6.0 for calculating interpolations and cross-sections of WRF-ARW 2D and 3D fields using lines and planes as slices.

20 February 2019

Added several examples showing how to use an updated version of gsn_coordinates to draw the edges and centers of hexagonal meshes. This capability was added in NCL V6.6.0:

  • MPAS - datagrid_9.ncl
  • ICON - datagrid_10.ncl
  • Geodesic - geo_660_1.ncl
